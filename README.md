## 項目介紹
這個項目是AI創作助手的前端項目，主要用來建立Ai創作助手的微信小程序客戶端頁面,
程序使用wxml、wxss、js技術，並使用WeChat小程序開發工具進行開發,實現了AI問答、標題生成等功能。

## 主要功能
- pages/index: AI智能問答頁面，提供輸入問題和生成答案功能
- pages/topic: 根據輸入文章, AI自動生成三個相關文章標題
